# 🎓 All Sciences

## تطبيق شامل للذكاء الاصطناعي في جميع مجالات العلوم

**All Sciences** هو تطبيق Electron متكامل يجمع كل أدوات العلوم في مكان واحد، مع دعم كامل للذكاء الاصطناعي، الكاميرا، الميكروفون، وـ 12 لغة عالمية.

---

## ✨ المميزات

### 🔬 8 مجالات علمية
- **🔢 الرياضيات** - آلة حاسبة علمية، حل معادلات، رسم بياني، مصفوفات، إحصاء
- **⚛️ الفيزياء** - محاكاة الحركة، دوائر كهربائية، موجات، بصريات، ثوابت
- **🧪 الكيمياء** - جدول دوري تفاعلي، موازن معادلات، كتلة مولية، محاكاة تفاعلات
- **🧬 الأحياء** - مستكشف الخلية، DNA/RNA، مربع بونيت، تصنيف، بيئات
- **🌌 الفلك** - النظام الشمسي، خريطة النجوم، تلسكوب، ثقوب سوداء
- **🏥 الطب** - جسم الإنسان، BMI، جرعات أدوية، فاحص أعراض
- **⚙️ الهندسة** - تحليل إنشائي، تروس، موائع، مواد، CAD
- **💻 الحاسوب** - محرر كود، خوارزميات، تشفير، SQL، شبكات

### 🤖 ذكاء اصطناعي
- مساعد AI متكامل مع ChatGPT/Claude/Gemini
- إدخال صوتي (Speech-to-Text)
- مسح المستندات بالكاميرا (OCR)
- ترجمة فورية بين اللغات

### 🌍 12 لغة
العربية، الإنجليزية، الإسبانية، الفرنسية، الألمانية، الصينية، الروسية، اليابانية، الهندية، البرتغالية، الإيطالية، التركية

### 📷 أجهزة
- وصول للكاميرا (مسح، تحليل صور)
- وصول للميكروفون (إدخال صوتي)
- صلاحيات آمنة على macOS/Windows/Linux

## 🚀 التثبيت السريع

### الطريقة 1: Quick Start

**Windows:**
```cmd
cd all-sciences
start.bat
```

**macOS/Linux:**
```bash
cd all-sciences
chmod +x start.sh
./start.sh
```

### الطريقة 2: يدوي
```bash
cd all-sciences
npm install
npm start
```

### الطريقة 3: بناء ملف تنفيذي
```bash
npm run build:win    # Windows .exe
npm run build:mac    # macOS .dmg
npm run build:linux  # Linux .AppImage
```
